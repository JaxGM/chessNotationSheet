Chess Notation Sheet

A Coding Project by Jax Miller

To Use

  • Type Your Moves Using Algebraic Notation (See Bellow)
  • Once the cell turns green, you are clear to move on.

Algebraic Notation

Pieces

  • K = King
  • Q = Queen
  • R = Rook
  • B = Bishop
  • N = Knight (I know)

Methods

  • Piece - End Location (Example: Qg5)
    • When moving a pawn, there is no need to specify P for Pawn, it's default.
  • Piece - Modifier Column - End Location (Example: N8e6)
    • This is used when two of the same piece can go to the same place.
  • Piece - Modifier Row - End Location (Example: Rbb3)
    • This is used when two of the same piece can go to the same place and are on the same column.

The Story

As a lover of chess and technology, writing game notation seemed so inefficient. I created this site to provide an easy and free way to write notation and export a PGN. To see the site's code or future features, check out our Github Page.
